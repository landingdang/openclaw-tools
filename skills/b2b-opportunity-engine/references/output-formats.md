## 4. 性格分析映射

### 性格标签

- `轻松随意`：发言更口语化，帖子轻松，愿意互动
- `正式专业`：表达克制，强调结构和专业度
- `关注创新`：频繁提新产品、新方法、新突破
- `关注效率`：高频提 ROI、效率、自动化、节省时间
- `关注技术`：强调架构、性能、技术细节
- `关注商业`：强调增长、收入、客户、案例

### 话术调整

- `轻松随意`：可轻微口语化，可少量 emoji，句子短
- `正式专业`：不用 emoji，语气稳，结构清楚
- `关注创新`：强调差异化、新做法、先发优势
- `关注效率`：强调节省时间、转化率、流程优化
- `关注技术`：强调性能、兼容性、实现细节
- `关注商业`：强调营收价值、客户案例、落地收益
```

## 5. 微信/企微添加话术模板

规则：
- `<= 60` 字
- 适用于微信/企业微信好友申请备注
- 必须提及对方公司、产品或近期动作
- 首次添加不要直接卖货

```text
[姓名]你好，看到你们在做[公司/动作]。我这边做[产品/能力]，能帮[这类团队]提升[具体价值]。想加个微信交流下。
```

## 6. 首次触达脚本模板

规则：
- `<= 120` 字
- 适用于脉脉私信、短信或电话开场白
- 结构：问候 -> 钩子 -> 价值 -> CTA
- CTA 保持轻量

```text
[姓名]你好，看到[公司/团队动作/招聘/采购信号]，所以来联系你。我们在帮[相似公司]解决[问题]，通常能带来[具体价值]。如果你愿意，我发你一个很短的案例，或者约 15 分钟交流下。
```

## 8. 缺失字段标记规则

- `confirmed`：已从公开或用户提供资料确认
- `inferred`：根据命名规则或上下文高概率推断
- `not_found`：当前未找到

对微信、企微、电话、资料页、职位等敏感字段，必要时写成：

```text
wechat_clue: not_found
phone: not_found
profile_url: inferred - [why]
```
