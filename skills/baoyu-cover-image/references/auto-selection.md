# 自动选择规则

当用户没有明确指定维度时，用本规则自动推荐。

## 推荐顺序

1. 先判断内容类型
2. 再判断标题信息量
3. 再判断适合的视觉强度

## 简化规则

| 场景 | 推荐 |
|---|---|
| 产品发布、公告、强曝光 | `hero` + 较强 `mood` |
| 技术、架构、概念文章 | `conceptual` |
| 观点、评论、引述型文章 | `typography` 或低图像密度方案 |
| 叙事、故事、人物内容 | `scene` |

## 默认值

- `aspect`: `16:9`
- `text`: `title-only`
- `mood`: `balanced`
- `font`: `clean`
