# 维度兼容性

不是所有组合都同样自然。

## 常见兼容关系

| 组合 | 结论 |
|---|---|
| `hero + vivid + digital` | 很稳 |
| `conceptual + cool + digital` | 很稳 |
| `scene + warm + painterly` | 很稳 |
| `typography + minimal` | 很稳 |
| `hero + pixel` | 谨慎使用 |
| `minimal + text-rich` | 容易冲突 |

## 原则

- 先保证信息传达，再追求风格奇观
- 如果组合本身冲突，优先牺牲风格感，不牺牲可读性
