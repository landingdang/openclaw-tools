# 风格预设

风格预设用于快速组合：

- `type`
- `palette`
- `rendering`
- `text`
- `mood`

## 适合的使用方式

当用户只说“给我做一个科技感封面”“要高级一点”“适合公众号头图”时，优先给预设，而不是逐项追问。

## 推荐策略

| 需求 | 推荐方向 |
|---|---|
| 科技/架构 | `conceptual + cool + digital` |
| 产品发布 | `hero + vivid + bold` |
| 观点/评论 | `typography + elegant` |
| 温暖叙事 | `scene + warm` |
