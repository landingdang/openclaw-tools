# 封面类型

## 常用类型

| 类型 | 适合场景 |
|---|---|
| `hero` | 需要强主视觉、强曝光的封面 |
| `conceptual` | 技术、架构、抽象概念 |
| `typography` | 观点、标题驱动型内容 |
| `metaphor` | 隐喻型表达 |
| `scene` | 叙事、人物、生活方式 |
| `minimal` | 极简风、低信息噪音 |

## 选择原则

- 技术内容优先 `conceptual`
- 情绪化和故事化内容优先 `scene`
- 观点型标题优先 `typography`
- 如果用户没说清，先给 2-3 套推荐
