# 风格库

这里只保留高层选择建议。具体子风格文件只在需要时再读。

## 常用风格

| 风格 | 适合内容 |
|---|---|
| `vector-illustration` | 知识解释、教程、技术内容 |
| `notion` | 简洁、知识型、SaaS 文档 |
| `elegant` | 商业、思辨、较正式内容 |
| `warm` | 个人成长、生活方式、教育 |
| `blueprint` | 架构、系统设计、技术图解 |
| `editorial` | 深度分析、媒体感图解 |

## 选择原则

- 技术文：`blueprint` / `vector-illustration`
- 教程文：`notion` / `vector-illustration`
- 叙事文：`warm`
- 商业分析：`elegant` / `editorial`
