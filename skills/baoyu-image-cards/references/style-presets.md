# 预设组合

`preset` 是 `style + layout (+ 可选 palette)` 的快捷写法。

## 推荐使用方式

- 用户没有明确风格时，优先推荐 preset
- 用户已经明确 `style` / `layout` 时，不必强推 preset

## 常见预设思路

| 预设方向 | 适合场景 |
|---|---|
| 知识卡 / 干货卡 | 概念总结、教程拆解、方法论 |
| 清单 / 排行 | 列点内容、工具推荐、步骤清单 |
| 对比卡 | A/B 对比、优缺点、方案选择 |
| 流程卡 | 教程步骤、上手流程、路线图 |
| 脑图卡 | 概念拆分、结构化知识 |

## 使用原则

- `preset` 先决定大的视觉方向
- `palette` 只做颜色覆盖
- 如果内容和 preset 明显不匹配，应退回手动指定 `style + layout`
